![Yii2-Admin](https://raw.githubusercontent.com/duiying/img/master/Yii2-Admin.png)

### 在线体验
暂不支持360极速浏览器兼容模式  
[Yii2-Admin](http://106.75.117.140)


### 安装
```
1. 克隆项目到本地
2. 修改 common/config/main-local.php 文件中的数据库配置
3. 新建数据库, 导入项目根目录下的sql文件
4. 访问后台 localhost/Yii2-Admin
    超级管理员: admin 123
    游客: youke 123
```
